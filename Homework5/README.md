## 清除屏幕
内存地址空间中，b8000h～bffffh 共32KB，是80x25彩色字符模式的显示缓冲区。
显示缓冲区分位8页，每页4KB，每页80x25=2000个字符，2000x2字节=4KB，共8页x4KB=32KB。
我们只在第0页操作
