### 格式化输出

使用INT 10H中断2号功能，移动光标

~~~
        mov ah,2 ;置光标位置 功能号
        mov bh,0 ; 页号
        mov dh,X ; 行号
        mov dl,Y ; 列号
        int 10h ;
~~~

### 32位整数除法
